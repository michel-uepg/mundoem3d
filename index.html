Instruções para Usar o VR

1. Use as setas do teclado para se mover no ambiente virtual.

2. Use o cursor vermelho para interagir com os objetos no ambiente.

3. Para selecionar um objeto, aponte o cursor e pressione a tecla "Espaço".

4. Mantenha-se atento ao tempo e à pontuação exibidos no topo da tela.

Olá! Bem-vindo! :)
Sobre O Projeto "Mundo em 3D": É uma iniciativa inovadora do curso de Engenharia de Software, da Universidade Estadual de Ponta Grossa (UEPG), situada no Paraná, voltada para a integração de tecnologias emergentes no ambiente educacional. Este projeto tem como objetivo principal explorar o potencial da Realidade Virtual (VR) como uma ferramenta pedagógica para enriquecer o processo de ensino-aprendizado em escolas da região. A Realidade Virtual é uma tecnologia que cria ambientes completamente digitais, permitindo que os estudantes sejam imersos em mundos virtuais onde podem interagir com objetos e cenários tridimensionais. Através deste projeto, a UEPG visa utilizar a VR para transformar a maneira como os conteúdos escolares são apresentados, proporcionando uma experiência educacional mais envolvente e interativa.
As tecnologias usadas neste projeto são HTML, CSS, JS, e framework A-Frame com VR para Web. Este projeto de extensão foi desenvolvido pelos acadêmicos Michel e Caroline, no 2º ano do curso de Engenharia de Software.
