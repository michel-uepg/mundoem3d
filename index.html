Sobre o projeto: Este projeto foi desenvolvido pelos acadêmicos Michel e Caroline da Universidade Estadual de Ponta Grossa (UEPG), como nota parcial do projeto de extensão 3, do segundo semestre do 2º ano do curso de Engenharia de Software.
O projeto é uma aplicação web baseada em HTML, CSS e JS, utilizando o framework A-frame. Construído o famoso Joguinho da Forca com realidade aumentada.
O objetivo desse projeto é estimular e facilitar os alunos das escolas a aprender um novo idioma de forma divertida e descontraída, utilizando tecnologias simples! :)
